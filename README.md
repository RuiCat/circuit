# circuit
Go实现的电气仿真,通过底层泛型与接口统一实现对 电子元件,气路元件,油路元件 综合综合仿真.通过事件同步实现 逻辑电路,潮汐计算 的联动仿真.

  > 需要将仿真器与前端隔离并且通过事件进行操作  
  > 也就是说前端通过通道传递事件到后端进行仿真器的控制
