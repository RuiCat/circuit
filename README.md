## 实现过程
  ## 基础结构实现
  > 初始化 [Context] 通过 [EventRegister] 注册元件  
  > 通过对元件实现 [Element] 封装为 [Event] 接口  
  ## 明确事件传递的作用
  > 后期元件列表由上下文管理,而元件的加盖过程需要实现并向处理所以通过事件传递来明确当前的迭代状态.  
  > 需要使用 [Flag] 进行事件同步
