## 介绍

求解代表电路的一组方程是直接的，即使不总是容易。然而，开发这组方程并不那么容易。两种常用的形成方程的方法是节点电压法（或节点法）和回路电流法（或网孔法）。我将简要描述每种方法，并提及它们的优点和缺点。如果您对这些方法有更全面的介绍感兴趣，我建议您在网上搜索适当的教程。最后我将讨论第三种方法，改进节点分析，它具有一些独特的优点。其优点之一是它适合算法求解——这些页面的最终目标是描述如何使用MATLAB程序生成代表电路的一组方程，这些方程可以符号求解。如果您只对使用该程序感兴趣，您可以直接使用上面的选项卡转到该页面。

本文讨论的电路是简单的电阻电路，具有独立电压源和电流源。受控源可以以直接的方式添加，但这里不考虑。

## 节点电压法

要将节点电压法应用于具有n个节点（有m个电压源）的电路，请执行以下步骤（根据Rizzoni）：

1. 选择一个参考节点（通常是地）。
2. 命名剩余的n-1个节点，并标记通过每个无源元件和每个电流源的电流。
3. 将基尔霍夫电流定律应用于每个未连接到电压源的节点。
4. 求解n-1-m个未知电压的系统。

### 示例1

考虑下面所示的电路

步骤1和2已经应用。要应用步骤3：
```
i1 + i2 = i3
i1 = (va - vb)/R1, i2 = (vc - vb)/R2, i3 = vb/R3
va = V1, vc = V2
所以 (va - vb)/R1 + (vc - vb)/R2 = vb/R3
(V1 - vb)/R1 + (V2 - vb)/R2 = vb/R3
```

在这种情况下只有一个未知数；代入数字并求解电路，我们得到vb = 24。

节点电压法通常直接应用，但如果一个或多个电压源不接地，就会变得有点困难。

考虑下面所示的电路。

显然这个电路与上面显示的电路相同，只是V1和R1交换了位置。现在我们写方程：
```
i1 + i2 = i3
i1 = -va/R1
i2 = (vc - vb)/R2
i3 = vb/R3
```

困难出现是因为电压源V1不再与其中一个节点电压相同。相反我们有：
```
vb - va = V1
vc = V2
或 va = vb - V1
所以 -va/R1 + (vc - vb)/R2 = vb/R3
(V1 - vb)/R1 + (V2 - vb)/R2 = vb/R3
```

注意最后一行与上一个电路相同，但为了求解电路，我们必须首先求解va。这个过程并不困难，但需要一点技巧，并且对于每个电路布局都会有点不同。处理这个问题的另一种方法是使用超节点的概念，这会使建立方程的规则复杂化（DeCarlo/Lin）。然而，超节点概念处理非接地电压源的情况，无需像我们在这里那样求解中间方程。

这里选择的示例很简单，但说明了节点分析的基本技术。它还说明了该技术的一个困难，即设置具有浮动电压源的方程。稍后介绍的改进节点分析技术，在遇到浮动电压源时也没有困难。

在上面显示的情况下，可以通过简单地反转R1和V1的顺序来创建解决方案。然而，我们正在尝试开发一种算法技术，因此我们不需要应用可能难以编程的见解。此外，像下面这样的电路（V1不接地），没有这样明显的简化。

## 网孔电流法

网孔电流法，毫不奇怪，与节点电压法相似。我们将"网孔"定义为原理图中的开放"窗口"。请注意，此方法需要一些修改才能与"非平面"电路一起工作（非平面电路是指那些不能绘制而没有电线交叉而不接触的电路）。以下规则遵循Rizzoni中的规则。

要将回路电流法应用于具有n个网孔（和m个电流源）的电路，请执行以下步骤：

1. 定义每个网孔及其对应的电流。使用一致的方法最容易，例如所有未知电流顺时针，所有已知电流遵循电流源的方向。
2. 将基尔霍夫电压定律应用于每个不包含电流源的回路。
3. 求解n-m个未知电压的系统。

### 示例3

考虑示例1中的电路，定义了网孔电流。

我们可以对两个回路应用KVL：
```
V1 - i1R1 - (i1 - i2)R3 = 0
(i1 - i2)R3 - i2R2 - V2 = 0
```

由于有两个方程和两个未知数，我们可以通过代入或矩阵方法求解。要通过矩阵方法求解，我们重写方程：
```
(R1 + R3)i1 - R3i2 = V1
-R3i1 + (R2 + R3)i2 = V2
或 [(R1 + R3) -R3; -R3 (R2 + R3)][i1; i2] = [V1; V2]
```

求解两个未知电流，我们得到：
```
i1 = 4
i2 = 1
```

虽然浮动电压源在使用节点电压法时往往会使电路方程的制定复杂化，但电流源或电压源的存在都不会使回路电流法复杂化。

## 一般评论
